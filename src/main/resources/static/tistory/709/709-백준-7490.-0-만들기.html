백트래킹 문제. 

숫자 사이에 +,-,' ' 중 한 가지를 넣고 계산하는데, 모든 경우의 수에 대하여 구하면 된다. 

 

나는 stringstream을 이용해 문자열을 만들고 공백을 제거하고, 또 다시 숫자를 추출해 계산하는 방식으로 구했다.

 

그런데 풀고 난 후 다른 분의 코드를 봤는데, 이렇게 하지 않고 재귀로 바로 계산되는 결과를 넘겨주는 방식으로 하니 훨씬 더 깔끔했다. 

바로 계산 결과를 넘겨주면 문자열 처리 부분이 불필요해진다.
