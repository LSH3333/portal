Heroku CLI:

  • Install the Heroku CLI if you haven't already.
  • Open a command prompt or terminal and log in to your Heroku account using the following command: 
  • Copy code
    heroku login
  • Once logged in, you can use the following command to open a PostgreSQL shell:
    Copy code
    heroku pg:psql
  • This will open a PostgreSQL shell where you can execute SQL queries. You can interact with your data by running SELECT queries, for example:
    sqlCopy code
    SELECT * FROM your_table_name;
  • Press Ctrl + D or type \q to exit the PostgreSQL shell.
