위의 LCS 만드는 문제는 길이만 구하면 됐지만 이번에는 실제 수열도 구해야한다.

실제 수열을 구할때는 역추적하면 된다.

 

d[i][j] == d[i-1][j] 일때는 i-1,j 로,

d[i][j] == d[i][j-1] 일때는 i,j-1 로,

else : 이때 해당 문자가 LCS 의 원소다. i-1,j-1 로
