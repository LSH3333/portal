백준 11047. 동전 0

이제부터는 그리디 문제들을 조금 집중적으로 풀어볼까 한다.

 

이 문제는 동전의 가치가 오름차순으로 주어지는데 입력을 받을때 내림차순이 되도록, 즉 가치가 큰 동전이 앞에오도록 입력받았다.

그 후 갖고있는 동전들중 큰 액수 동전부터 탐색하면서 목표금액 k보다 작은 동전이 나오면 k에서 빼주는 방식으로 

k가 0이되면 while문이 종료되도록 했다.

동전의 가치가 내림차순으로 정렬되어 있기 때문에 k보다 작은 액수중 가장 큰 동전이 무조건 선택된다.

 

그리디 알고리즘은 그때그때 가장 최고라고 생각되는 선택을 하는것으로 

k보다 작은 동전중 가장 큰 동전을 계속 선택하는 전형적인 그리디 알고리즘을 이용해 푸는 문제이다.

 

#include <iostream>
using namespace std;

int n, k;
int coin[10];

int main()
{
    cin >> n >> k;
    // 내림차순 정렬 되도록
    for(int i = n-1; i >= 0; i--)
    {
        cin >> coin[i];
    }

    int cnt = 0;

    while(k > 0)
    {
        // 갖고있는 동전들중 큰 액수 동전부터 탐색하면서 k보다 작은 동전이 나오면 k에서 뺀다
        // 동전이 내림차순으로 정렬되어있으므로 k보다 작은 액수중 가장 큰 동전이 무조건 선택된다
        for(int i = 0; i < n; i++)
        {
            if(coin[i] <= k)
            {
                cnt++;
                k -= coin[i];
                break;
            }
        }
    }

    cout << cnt;
}
