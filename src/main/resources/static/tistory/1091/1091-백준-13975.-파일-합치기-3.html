주어진 예제가 절묘하게 악랄했던 문제였다 ㅋㅋ.

이 문제는 (두 파일의 합) 을 계속 더해나갔을때 최소값을 찾아야 하기 때문에 가장 크기가 작은 파일 두 개를 골라서 합해주면 된다.

따라서 min heap 우선순위 큐에 모두 넣고 가장 작은 두 개의 파일을 빼서 합한다음 다시  우선순위 큐에 넣어주면 된다.

 

그런데 주어진 예제가 {40, 30, 30, 50} 이고 문제에서 (40+30)=70 + (30+50)=80 + (70+80)=150 = 300 으로 주어진다.

이 경우 위에 설명한 알고리즘대로 하지 않았는대도 최소 값인 300이 나와버리기 때문에 문제가 햇갈려진다.
