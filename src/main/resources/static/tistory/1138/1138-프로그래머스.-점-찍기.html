x = 0 일때 y 의 최대값은 d - (d % k) 이다.  (d 가 아님!)

x = 0 일때 가능한 점의 갯수는 (y / k + 1) 이다.

 

이제 x 좌표를 k 만큼 늘리면 당연히 (x,y)는 d 보다 크거나 같아질것이다.

따라서 새로운 x 좌표의 y 값을 구하기 위해서 거리가 d 보다 작거나 같아질때까지 y 에서 k 를 빼면 된다.
