배낭 문제.

 

배낭 문제의 핵심은 물건을 하나씩 늘려가면서, 해당 물건을 넣을지 안넣을지 판단하는 것이다.

 

// i번째 물건까지 확인했을때, j 일 까지 읽을수 있는 최대 페이지 수
int d[21][210];

점화식:

d[i][j] = max(d[i - 1][j], d[i - 1][j - day[i]] + page[i]);
