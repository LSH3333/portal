백준 5569. 출근 경로

PS

2023-09-09 14:56:55

https://www.acmicpc.net/problem/5569

 

이 문제의 핵심은 교차로를 연속으로 돌 수 없다는 것이다.

이 조건이 없다면 점화식 d[i][j] = d[i-1][j] + d[i][j-1] 인 간단한 문제지만 위 조건 때문에 어려워진다.

 

결국 문제는 이전에 교차로를 돌았는지 안돌았는지를 판단해야하기 때문에 다음과 같이 4가지를 고려해야 한다.

 

1) d[i][j][0] = 이전에 동쪽 방향으로 현재 지점에 도달함 

2) d[i][j][1] = 이전에 북쪽 방향으로 현재 지점에 도달함

3) d[i][j][2] = 북쪽 이동, 동쪽 이동 교차로로 현재 지점에 도달함 

4) d[i][j][3] = 동쪽 이동, 북쪽 이동 교차로로 현재 지점에 도달함 

 

1) 의 경우 동쪽 방향으로 현재 지점에 도달했기 때문에 다음으로 동쪽, 북쪽 모두 이동가능하다

2) 의 경우 북쪽 방향으로 현재 지점에 도달했기 때문에 다음으로 동쪽, 북쪽 모두 이동가능하다

3) 의 경우 북동 교차로로 현재 지점에 도달했기 때문에 다음으로 동쪽으로만 이동 가능하다

4) 의 경우 동북 교차로로 현재 지점에 도달했기 때문에 다음으로 북쪽으로만 이동 가능하다

 

3)을 예로 설명해보면 북동 교차로로 현재 지점에 도달했다는 것은, 다음으로 북쪽으로 이동하게 된다면 교차로를 연속으로 2번 방향을 바꾼것이 된다.
