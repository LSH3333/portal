유니티

오브젝트(Bomb)는 날린 일정 시간 후 파괴하고

trail renderer는 남기고 싶어서 사용한 방법:

Bomb의 child 오브젝트를 만들고 그곳에 trail renderer 컴포넌트를 생성한 후

 

Bomb 오브젝트가 Destroy될때 Parent와 Child를 분리 시킴.

 

void OnDestroy()
    {
        _trail.transform.parent = null;
    }

 

이렇게 하면 Bomb이 destroy될때 Parent와 Child가 분리되면서 Child는 남게되므로

trail renderer 도 유지됨.
