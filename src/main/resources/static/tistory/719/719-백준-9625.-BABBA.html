백준 9625. BABBA

화면의 B는 BA로 

화면의 A는 B로 

 

버튼을 눌렀을때 A가 되는 경우는 B->BA

현재 화면의 A의 갯수는, 버튼 누르기 이전의 B의 갯수.

 

버튼을 눌렀을때 B가 되는 경우는 B->BA, A->B

현재 화면의 B의 갯수는, 버튼 누르기 이전의 A갯수와 B의 갯수의 합.

 

점화식:

A[i] = B[i-1]

B[i] = A[i-1] + B[i-1]
