나가기전 시간이 애매하게 남아서 간단한 문제 하나 풀어봤다.

 

1. 벡터에 0을 푸쉬한다. (버섯을 하나도 안먹은 경우 계산을 위해)

2. 벡터에 버섯들의 점수를 푸쉬한다. 

3. 각각 버섯을 0번째 까지 먹었을때, 1번째 까지 먹었을때, ... , 10개를 모두 먹었을때 점수와 100의 차 (절댓값)을 구한다

4. 차가 가장 작을때, 점수가 가장 높을때를 갱신한다

 

#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

vector<int> mushroom;
int ans = 0;

int main()
{
    mushroom.push_back(0);
    for(int i = 0; i < 10; i++)
    {
        int score;
        cin >> score;
        mushroom.push_back(score);
    }

    int minSub = 99999;
    for(int i = 0; i < mushroom.size(); i++)
    {
        int sum = 0;
        for(int j = 0; j <= i; j++)
        {
            sum += mushroom[j];
        }

        int sub = abs(100-sum);
        if(sub <= minSub && sum >= ans)
        {
            minSub = sub;
            ans = sum;
        }
    }
    cout << ans;
}
