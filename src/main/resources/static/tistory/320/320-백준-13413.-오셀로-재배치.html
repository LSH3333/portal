주어진 문자열 initial과 목표 문자열 goal을 비교하면서 서로 엇갈린 부분이 몇개인지 찾는다.

예를들어 initial과 goal이 다음과같다면

B B W W B B

W W B B W B

 

initial이 b고 goal이 w인 경우 bw = 3

initial이 w고 goal이 b인 경우 wb = 2

 

bw와 wb중 작은 수만큼 1번 연산을 해줄수 있고, 나머지는 2번연산으로 처리한다.

 

#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);

    int tc;
    cin >> tc;

    while(tc--)
    {
        int n;
        string initial, goal;
        cin >> n >> initial >> goal;

        int bw = 0, wb = 0, none = 0;
        for(int i = 0; i < n; i++)
        {
            if(initial[i] == 'B' && goal[i] == 'W')
                bw++;

            else if(initial[i] == 'W' && goal[i] == 'B')
                wb++;
            else if(initial[i] != goal[i])
                none++;
        }

        int minCnt = min(bw,wb);
        int maxCnt = max(bw,wb);

        int swapCnt = minCnt;
        int left = maxCnt - swapCnt;
        int ans = swapCnt + left + none;

        cout << ans << '\n';
    }



}
