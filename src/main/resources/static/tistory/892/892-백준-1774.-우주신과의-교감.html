백준 1774. 우주신과의 교감

PS

2022-09-26 13:03:58

모든 좌표들끼리의 거리를 구하고 MST를 만들면 된다.

노드의 갯수는 최대 1000개 이고 모든 노드들을 이으면 간선의 갯수는 1000 + 999 + ... 1 + 0 이 된다.

등차수열의 식으로 계산해보면 (1000 * 999) / 2 = 약 50만 이므로 충분하다. 

 

다른점은 이미 이어져 있는 노드들이 주어진다는 것이다.

해당 노드들은 미리 Union처리 해주고 Kruskal 알고리즘을 돌리면 된다.
