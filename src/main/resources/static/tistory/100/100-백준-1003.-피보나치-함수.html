백준 1003. 피보나치 함수

PS

2021-03-06 17:05:32

www.acmicpc.net/problem/1003

 

처음에 시간제한 보고 그냥 주어진 코드에 memoization 추가하면 될줄 알았는데

 

// memoization!!
if(d[n] > 0) { return d[n]; }

 

이렇게 하게되면 0과 1의 호출이 생략되기 때문에 호출 횟수를 출력할수가 없다.

 

그래서 memoization을 수정해야 하나 이리저리 고민하다가 0과 1의 호출 횟수를 세어보니

 

fibo 0 호출 횟수 1 호출 횟수
fibo(0) 1 0
fibo(1) 0 1
fibo(2) 1 1
fibo(3) 1 2
fibo(4) 2 3
fibo(5) 3 5

 

그냥 0과 1의 호출횟수도 피보나치 수열이었다.

idx 0 1 2 3 4
fibo 0 1 1 2 3

fibo(3)의 0의 호출 횟수 : 1

fibo(3)의 1의 호출 횟수 : 2

 

fibo(4)의 0 호출 횟수 : 2

fibo(4)의 1 호출 횟수 : 3

 

fibo(n)의 0 호출 횟수 : fibo(n-1)

fibo(n)의 1의 호출 횟수 : fibo(n)

 

따라서 그냥 피보나치 Top-down 코드를 그대로 쓰면된다.

 

#include <iostream>
using namespace std;

int d[101] = {0,1,};

int fibonacci(int n) {
    // base
    if(n == 0) {   return 0; }
    if(n == 1) {   return 1; }

    // memoization!!
    if(d[n] > 0) {  return d[n]; }


    d[n] = fibonacci(n-1) + fibonacci(n - 2);
    return d[n];
}

int main() {
    int t;
    cin >> t;

    while(t--) {
        int n;
        cin >> n;
        fibonacci(n);

        if(n == 0) cout << "1 " << "0" << '\n';
        else if(n == 1) cout << "0 " << "1" << '\n';
        else cout << d[n-1] << ' ' << d[n] << '\n';

    }
}
