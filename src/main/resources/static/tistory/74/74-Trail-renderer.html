오브젝트가 이동했을때 오브젝트 뒤에 포물선처럼 트레일이 남음.

 

Material : Element를 설정해서 다른 종류의 trail을 남길수 있음.

Time : Trail이 생성된후 언제 사라질 것인가 (0, Infinity)

Min Vertex Distance : 작을수록 trail이 각지고, 클수록 둥글어짐.

 

Width : Trail의 너비

그래프를 이용해 trail이 언제 너비가 작아지고 커질지 조절할수 있음.
