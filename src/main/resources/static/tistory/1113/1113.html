spring project heroku에 배포시 경로 문제 (template might not exist or might not be accessible by any of the configured Template Resolvers)

Web

2023-06-03 14:36:35

spring 기반 web 을 heroku 에 배포했는데 특정 경로에 접근했을때 다음 에러가 떴다.

 

template might not exist or might not be accessible by any of the configured Template Resolvers

 

알고보니 절대/상대 경로 문제였다. 리턴하는 뷰 이름이 상대경로 예를들어 return "/members/register" 이런식이면 경로 문제가 생긴다.
절대 경로로 return "members/register" 로 고쳐주니 해결.
