백준 19939. 박 터뜨리기

모든 바구니에는 1개 이상의 박이 들어가야하고, 같은 수의 박이 들어있으면 안되므로

먼저 박을 1,2,3, ... , k 개씩 분배한다.

만약 1,2,3,...,k 의 합이 n이 넘어간다면 불가능한 경우 이므로 -1을 출력하고 종료한다.

 

가능한 경우는 두가지 경우가 있는데 1,2,3,...,k개씩 분배한후에 1)모두 같은수의 박을 넣을수 있는 경우와,

2)그렇지 않은경우다.

 

1) 

n=6, k=3

1 2 3

 

n=15, k=3

1 2 3 분배한후

2 3 4   // sum=9

3 4 5  // sum=12

4 5 6  // sum=15

 

2)

n=13, k=3

1 2 3  // sum=6

2 3 4  // sum=9

3 4 5  // sum=12, 모든 바구니는 다른 수의 박을 가져야하므로 마지막 박에 박 분배

4 5 7  // sum=13

 

1번의 경우 첫박과 마지막박의 차는 k-1인것을 볼수있고

2번의 경우 첫박과 마지막박의 차는 k인것을 볼수 있다.

 

 

#include <iostream>
using namespace std;

int n,k;

int main()
{
    cin >> n >> k;


    int sum = 0;
    // 박을 1,2, ... , k 개 씩 분배한다
    for(int i = 1; i <= k; i++)
    {
        sum += i;
        // 불가능 조건
        if(sum > n)
        {
            cout << -1;
            return 0;
        }
    }
    // 분배한 박의수 뺌
    n -= sum;

    // 1,2,...,k 개씩 분배한 후, 모든 박에 동일한 박의 수를 더할수 있는경우
    if(n % k == 0)
    {
        cout << k-1;
    } // 균등하게 더할수 없는 경우
    else
    {
        cout << k;
    }

}
