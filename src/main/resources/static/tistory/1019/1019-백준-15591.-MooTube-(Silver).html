그래프를 만든 후 깊이 우선 탐색으로 엣지의 최소값을 기억하면서 int mark[r][c] 2차원 배열에 USADO(r,c) 를 저장한다. 

깊이 우선 탐색은 1~N 노드들에 대하여 진행하는데 mark 배열의 값에 따라 탐색한곳은 다시 방문하지 않도록 하면 시간초과는 나지 않는다. (만약 새로운 dfs 를 진행할때마다 방문 여부 배열을 초기화해서 진행하면 시간초과 날 수도? )
