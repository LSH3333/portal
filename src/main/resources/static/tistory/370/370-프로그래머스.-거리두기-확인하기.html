places의 크기가 5고, 만들어지는 그래프도 크기가 5x5이기 때문에 중첩 반복문으로 풀었다.

대기실들을 그래프로 만들고 (0,0) 부터 (5,5)까지 순회하면서 거리가 2이하인 사람들이 있다면 다음을 확인한다.

 

두 사람의 좌표가 (i,j), (a,b)일때 

1. 두 사람이 대각선상에 있다면 (i != a && j != b)

   파티션을 사이에 두고 앉았는지 확인. (v[i][b] == 'X' && v[a][j] == 'X')

 

2. 두 사람이 일직선상에 있다면 

   두 사람 사이에 파티션(X)가 있는지 확인. (v[(i+a)/2][(j+b)/2)] == 'X')
