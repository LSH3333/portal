이 문제는 일단 memoization을 이용해 재귀함수의 시간을 줄이는 문제인데,

처음에 신경 써야할 부분이 두 군데 있다고 느꼈다.

 

1. 함수에 memoization 추가

2. 함수의 input인 a,b,c가 -50 <= a, b, c <= 50 이므로 음수값이 들어오는 것을 고려해야한다.

 

그런데 주어진 함수를 보면 조건중에 a or b or c 중에 하나라도 0이하면 어처피 1을 리턴하기 때문에 음수값은 고려하지 않아도 된다는것을 알게됐다. 

그래도 배열에 음수 인덱스에 대해 궁금해져서 찾아봤다.

(글: tose33.tistory.com/102)

 

따라서 이 문제는 그냥 3차원 배열 선언하고, memoization만 추가해주면 되는 간단한 문제였는데..

 

 

아무리봐도 맞았는데 계속 틀렸다고 나왔다.. 뭔가 특별한 조건이 있나 싶어서 계속 들여다봤지만.

알고보니까 답이 

w(1, 1, 1) = 2

w(2, 2, 2) = 4

w(10, 4, 6) = 523

w(50, 50, 50) = 1048576

w(-1, 7, 18) = 1

이런 식으로 앞에 w(a,b,c) =  ... 이것까지 출력해야되는데 제대로 안보고 그냥 우변의 답만 출력해서 틀린것이었다. 내시간.

 

#include <iostream>
using namespace std;

long long mem[110][110][110];

long long recur(long long a, long long b, long long c) {
    if(a <= 0 || b <= 0 || c <= 0) { return 1; }

    if(a > 20 || b > 20 || c > 20) {  return recur(20,20,20); }
    if(mem[a][b][c] != 0) {
        return mem[a][b][c]; // memoization
    }


    if(a < b && b < c) { mem[a][b][c] = recur(a,b,c-1) + recur(a,b-1,c-1) - recur(a, b-1, c); }
    else
    {
        mem[a][b][c] = recur(a-1,b,c) + recur(a-1, b-1, c) + recur(a-1,b,c-1) - recur(a-1, b-1,c-1);
    }
    return mem[a][b][c];


}

int main() {
    while(true) {
        long long a, b, c;
        cin >> a >> b >> c;

        if(a == -1 && b == -1 && c == -1) break;

        cout << "w(" << a << ", " << b << ", " << c << ") = " << recur(a,b,c) << '\n';
    }
}
