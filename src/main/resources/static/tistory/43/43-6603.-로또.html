next_permutation(), prev_permutation 함수를 이용.

next_permutation을 이용하는 새로운 방법을 이 문제에서 알게되었다.

 

문제는 k(k>6)개의 숫자중 6개를 뽑아서 순열을 만드는 거라고 볼수 있다.

그럼 예제에 있는것처럼 k = 7일때는 7개중 6개를 뽑아야한다.

 

[1,1,1,0]을 prev_permutation을 돌려보면,

 

[1,1,1,0] 

[1,1,0,1]

[1,0,1,1]

[0,1,1,1]

 

이 나온다.

 

이것은 4개의 숫자 중 3개를 뽑아서 순열을 만드는 것과 같다.

숫자가 [1,2,3,4]라면

 

[1,1,1,0]  [1,2,3,4]

[1,1,0,1]  [1,2,3,4]

[1,0,1,1]  [1,2,3,4]

[0,1,1,1]  [1,2,3,4]

 

 

#include <iostream>
using namespace std;
#include <algorithm>

int main() {
	while(true) {
		int n;
		cin >> n;
		if (n == 0) break; // n = 0이면 프로그램 종료

		int a[100];
		int b[100] = { 0 };

		for (int i = 0; i < n; i++) {
			cin >> a[i];
		}

		for (int i = 0; i < 6; i++) { // 6개를 골라야 하므로 6개의 1
			b[i] = 1;
		}


		do {

			for (int i = 0; i < n; i++) {
				if (b[i] == 1) cout << a[i] << " ";;
			}
			cout << "\n";
		} while (prev_permutation(b, b + n));
	
		cout << "\n";
	}
}
