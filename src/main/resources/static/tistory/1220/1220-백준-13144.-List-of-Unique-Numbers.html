이 문제는 현재 뽑은 연속되는 수를 기록하면서 r을 증가시키고, 중복되는 숫자와 만나면 멈추고 l 을 증가시켜야 한다.

중요한건 l 을 증가시키면서 지나온 숫자들은 제외된다는 것이고, 그러면 이후로 r을 증가시키면서 만들어지는 선택한 수의 집합은 이전에 없던 새로운 집합이라는 것이다.

 

 

l = 0, r = 0 부터 시작해서 중복되는 값이 나올때까지 r 을 증가시킨다.

 

중복되는 값이 나오면 answer += (r-l) 이다.

l 값을 증가시킨다.

여기서 l 값을 증가시키기 전에 visited[l] = false 로 해줘야 한다.
