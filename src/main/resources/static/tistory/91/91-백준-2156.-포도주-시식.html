d[n]을 포도주의 잔 수가 n개 일때, 최대로 마실 수 있는 포도주의 양 이라고 하자.

예제는 n = 6, 

포도주의 양은 6, 10, 13, 9, 8, 1

 

wine[1] wine[2] wine[3] wine[4] wine[5] wine[6]
6 10 13 9 8 1

 

d[1]은 와인잔이 한잔 이므로 그냥 wine[1]

d[1] = wine[1]

 

d[2]는 두잔 까지는 연속으로 마실수 있으므로 

d[2] = wine[1] + wine[2]

 

d[3] 부터는 포도주를 연속으로 두잔까지만 마실수 있기 때문에 생각을 해야한다.

세번째 잔을 마실때 내가 취할수 있는 행동은 

1. 세번 째 잔을 안마신다

2. 세번 째 잔을 마신다 

이때 세 번째 잔을 마신다면, 두번 연속으로까지만 마실수 있으므로

2-1. wine[1] + wine[3] 이거나

2-2. wine[2] + wine[3] 이다. 

 

 

1번 case

세 번째 잔을 안 마실 경우 그 전까지 내가 최대로 마신 양 즉 

d[3] = d[2]가 된다. 

 

2-1 case

세 번째 잔을 마시는데 첫 번째잔 마시고 두번째를 건너뛰고 세번째 잔을 마신다.

그러면 첫번째 잔까지 마신 최대의 양 d[1] 더하기

세 번째 잔 wine[3] 이 답이다.

d[3] = d[1] + wine[3]

 

2-2 case

마찬가지로 세번째 잔을 마시는데 이번에는 바로 전 잔인 두번째잔과 세번째 잔을 연속으로 마신다.

그런데 두번 연속으로까지만 마실수 있으므로 첫번째 잔은 무조건 안마셨어야 한다.

따라서 0번째 잔까지 마신 최대 양 d[0] + wine[2] + wine[3]

즉 d[3] = d[0] + wine[2] + wine[3]

 

위 의 세가지 케이스 중 가장 큰 값이 d[3]이다.

 

#include <iostream>
using namespace std;
#include <algorithm>

int wine[10001];
int d[10001];

int main() {
    int n;
    cin >> n;

    for(int i = 1; i <= n; i++) cin >> wine[i];

    d[0] = 0;
    d[1] = wine[1];
    d[2] = wine[1] + wine[2];

    for(int i = 3; i <= n; i++) {
        d[i] = max({d[i-1], d[i-2]+wine[i], d[i-3]+wine[i-1]+wine[i]});
    }

    cout << d[n];
}
