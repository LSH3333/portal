로프의 길이를 오름차순으로 정렬해서

짧은 로프부터 현재 선택한 로프 길이를 기준으로  n-i개를 병렬 연결했을 때 견딜수 있는 중량의 최댓값을 갱신한다.

 

예를들어 입력이 다음과 같이 주어지면

4     

10   // 10x4=40 

15   // 15x3=45

20  // 20x2=40

30 // 30x1=30 

 

이중 최댓값은 15x3=45

 

#include <iostream>
#include <algorithm>
using namespace std;

int n;
int ans = 0;
int rope[100001];

int main()
{
    cin >> n;
    for(int i = 0; i < n; i++)
        cin >> rope[i];

    sort(rope, rope+n);

    for(int i = 0; i < n; i++)
    {
        int res = rope[i] * (n-i);
        ans = max(ans, res);
    }
    cout << ans;

}
