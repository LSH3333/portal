백준 2666. 벽장문의 이동

PS

2023-01-30 15:58:31

https://www.acmicpc.net/problem/2666

 

두 개의 문만이 열려있다.

그렇다면 우리가 매순간 선택할수 있는 선택지는 두가지 뿐이다.

 

열려있는 문을 i, j라고 하고 사용할 벽장문의 번호를 n이라고 할때 

i문을 닫고 n 문을 여는 경우, j문을 닫고 n 문을 여는 경우 두가지 뿐이다. 

직접 그려보면 알겠지만 i문을 닫고 n문을 여는 경우 문의 이동 횟수는 abs(i-n)이다. 

 

따라서 점화식은 다음과 같다 

dp[depth+1][n][j] = min(dp[depth+1][n][j], dp[depth][i][j] + abs(n-i));
dp[depth+1][i][n] = min(dp[depth+1][i][n], dp[depth][i][j] + abs(n-j));
