배낭 문제인데 약간의 변형이 있다.

기본적으로 배낭 문제의 점화식은 거의 다음과 같다

d[i][j] = max(d[i-1][j], d[i-1][j-cost[i]] + people[i]) 

즉 i번째 요소를 선택할지 안할지 아니면 i-1번째 까지 계산한 값을 유지할지다.

 

그런데 이 문제는 (3, 5) 가 주어지면 3원으로 5명의 고객을 유치 가능한데 여기에 더해서 6원으로 10명, 9원으로 15명 .. 이런식이다.

따라서 이 문제의 점화식은 

d[i][j] = max(d[i-1][j], d[i][j-cost[i]] + people[i]) 가 된다.

 

j의 최댓값이 햇갈릴수 있는데 주어지는 비용,고객수가 (1, 100)일때를 생각해보면 j의 최댓값은 100000이다.
