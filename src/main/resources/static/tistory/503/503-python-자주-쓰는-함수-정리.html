Dictionary get 함수 

c++에서는 key값이 없으면 0을 반환하지만 파이썬에서는 그냥 에러를 발생시킨다.

get 함수를 사용하면 에러를 발생시키지 않고 none 값을 반환함으로서 예외처리를 할 수 있다.

m = {}
m[0] = 1
if m.get(10) is None: print('none')
