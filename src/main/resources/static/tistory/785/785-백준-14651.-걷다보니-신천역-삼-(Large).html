0,1,2 로 3의 배수인 길이 N인 숫자의 갯수를 구하는 문제다.

3의 배수에 특징이 었었던것 같은데 기억이 안나서 블로그를 뒤져봤는데 있었다.

 

https://tose33.tistory.com/464

 

3의 배수의 특징은 각 자리의 숫자를 모두 더했을때 합이 3의 배수이다. 

 

근데 이 문제는 그냥 패턴이 눈에 보여서 그걸로 풀었다.

N = 1 일때 0 

N = 2일때 2

N = 3 일때 6 

N = 4 일때 18 ... 

 

3배씩 늘어나는것을 알 수 있다.
