백준 10942. 팰린드롬?

PS

2023-08-14 14:06:42

https://www.acmicpc.net/problem/10942

 

질문의 갯수가 100 만개 이므로 단순히 각 질문마다 탐색하면 TLE 일 것이다.

 

[s,e] 가 팰린드롬이려면 [s+1, e-1] 도 팰린드롬이어야 한다.

즉 [s,e] 가 팰린드롬이려면 (arr[s] == arr[e] && palindrome[s+1][e-1]) 이어야 한다. 

 

top-down dp 방식으로 풀어봤다.

top down 에서 시간을 절약하기 위해선 palindrome[s][e] 를 이미 탐색했으면 바로 해당 값을 리턴해주면 된다.
