이차원 다이나믹으로.

d[n][a] : 길이가 n인 계단수 중 마지막 수가 a인 총 갯수

 

계단수란 1씩 차이나는 수 이다.

d[3][2]는 길이가 3인 계단수 중 마지막 수가 2인 총 갯수.

마지막 수가 2려면 그 전 자리가 1 or 4여야 하기 때문에

d[3][2] = d[2][1] + d[2][3]

 

d[5][3]는 길이가 5인 계단수 중 마지막 수가 3인 총 갯수.

마지막 수가 3려면 그 전 자리가 2or 4이기 때문에 

d[5][3] = d[4][2] + d[4][4]

 

따라서 점화식은

d[n][a] = d[n-1][a-1] + d[n-1][a+1] 이다.

 

 

하지만 예외가 있다.

마지막 수가 0 or 9 일때이다.

 

마지막 수가 0 이면 전 자리에서는 1밖에 올수가 없다.

마지막 수가 9 이면 전 자리에서는 9밖에 올수가 없다.

 

따라서 이 둘만 예외로

d[n][0] = d[n-1][1]

d[n][9] = d[n-1][8]

 

#include <iostream>
using namespace std;
#define MOD 1000000000
long long d[101][11];

int main() {
    int n;
    cin >> n;

    d[1][0] = 0;
    for(int i = 1; i < 10; i++) d[1][i] = 1;

    for(int i = 2; i <= n; i++) {
        for(int a = 0; a <= 9; a++) {
            if(a == 0) d[i][0] = d[i-1][1] % MOD;
            else if(a == 9) d[i][9] = d[i-1][8] % MOD;
            else {
                d[i][a] = (d[i-1][a-1] + d[i-1][a+1]) % MOD;
            }
        }
    }


    long long ans = 0;
    for(int i = 0; i < 10; i++) ans += d[n][i];
    cout << ans % MOD;


}

숫자가 엄청 크기 때문에 자료형에 주의해야하고, 

배열 d에 저장 할때도 1000000000로 나눠서 저장해야한다.
