백준 13460. 구슬 탈출 2

PS

2022-05-19 20:12:21

https://www.acmicpc.net/problem/13460

 

보드를 4방향으로 움직일수 있는데 보드를 움직여 최소 몇번만에 빨간 구슬만을 구멍으로 빼낼수 있는지 구하는 문제다.

보드를 움직이는건 dfs로 순열을 만들듯이 만들어주면 된다.

문제의 핵심은 보드를 움직이는 기능의 구현이었다.

 

나는 네 방향 움직이는걸 모두 직접 구현했다. 

거의 기능이 비슷해서 머리를 잘 굴리면 하나의 함수로 합칠수 있을것 같긴한데, 이 부분은 나중에 다른 분들의 답을 참고해야겠다.

 

두 구슬은 같은 좌표에 존재할수 없기 때문에 만약 두 구슬이 같은 행or열에 있다면 각 방향마다 먼저 움직여야 하는 구슬이 다르다.

예를들어 보드를 왼쪽으로 기울때 두 구슬이 같은 행에 있다면, 왼쪽에 있는 구슬이 먼저 움직여야 한다.
