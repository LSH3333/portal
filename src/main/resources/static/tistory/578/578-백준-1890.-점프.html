시작지점이 아닌 도착지점에서부터 계산해본다.

d[i][j]가 (i,j)칸에서 도착지점 까지 갈수 있는 경우의 수라고 한다면, 

d[i][j]는 (i,j)칸에서 현재 칸에 적혀진 숫자만큼 오른쪽으로 떨어진 칸이 도착지점 까지 갈 수 있는 경우의 수 + 

(i,j)에서 현재 칸에 적혀진 숫자만큼 아랫쪽으로 떨어진 칸이 도착지점 까지 갈 수 있는 경우의 수 이다.

 

이렇게 계산할때 현재 칸에 적혀진 숫자만큼 떨어진 곳이 주어진 범위 N을 벗어날수 있기 때문에 처음에 배열의 크기를 넉넉히 잡아주면 boundary를 신경쓰지 않아도 된다.

 

점화식은 

// (i,j)에서 오른쪽으로 숫자만큼 떨어진곳, 아랫쪽으로 숫자만큼 떨어진곳 

d[i][j] = d[i][j + distance] + d[i+distance][j]
