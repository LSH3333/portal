최대 노드의 갯수가 1만개로 1만 x 1만 = 1억, 주어진 시간이 5초로 그냥 모든 노드에 대해서 이동가능한 노드를 세어주면 되는 문제였다.

 

그런데 처음에 1만 x 1만 = 10억으로 실수로 잘못 계산해서;; 그냥 모든 노드 탐색하면 안될거같아서 dfs, dp 를 이용해서 풀었다.

결론적으로 이 문제는 두 노드가 사이클이 생길수 있어서 dp로 풀수 없는 문제다;

그냥 노드 관계 만들어놓고 모든 노드를 탐색하면 풀수있다.
