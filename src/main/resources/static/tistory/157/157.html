오브젝트의 position, rotation 고정시키기 (Rigidbody Freeze)

유니티

2021-05-13 16:57:09

Rigidbody가 부착된 오브젝트의 position이나 rotation을 고정시킬때 

Rigidbody의 Constraints의 Freeze Position, Freeze Rotation을 사용할수 있다.

 

 

그런데 이 오브젝트에게 child collider가 있으면 rigidbody가 계산을 다시해서 적용이 안된다.

이럴때는 스크립트에서 LateUpdate같은곳에서 명시해주면된다.

private void LateUpdate()
    {
        transform.localEulerAngles = new Vector3(0, transform.localEulerAngles.y, transform.localEulerAngles.z);
    }
