백준 2617. 구슬 찾기

2023-02-17 14:10:45

https://www.acmicpc.net/problem/2617

 

어떤 구슬이 무게가 중간이 될 수 없다는 것은, 해당 구슬 보다 가벼운 구슬이 N/2+1 개 이상이거나, 무거운 구슬이 N/2+1 개 이상이라는 뜻이다.

 

숫자가 클수록 무거운 다섯개의 구슬을 생각해보자. 

1 2 3 4 5 

3번 구슬이 중간 구슬이며, 중간 구슬 보다 무거운 구슬은 2개, 가벼운 구슬은 2개다.

2번 구슬보다 무거운 구슬은 3개이며 가벼운 구슬은 1개 이므로, 2번 구슬은 중간 구슬일수 없다. 

 

서로 반대 방향인 간선을 저장해서 (가벼운쪽, 무거운쪽) dfs나 bfs로 해당 구슬보다 가벼운 구슬의수, 무거운 구슬의 수를 카운팅한다.

가벼운 구슬의 수나 무거운 구슬의 수가 N/2+1 이상이라면 해당 구슬은 중간 구슬일 수 없다.
