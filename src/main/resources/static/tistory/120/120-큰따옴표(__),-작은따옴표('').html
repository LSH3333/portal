큰따옴표(""), 작은따옴표('')

노트

2021-03-20 15:25:02

c, c++에서 작은따옴표는 single character를 나타내고, 큰따옴표는 string literal을 나타낸다.

'a'는 그냥 single character 지만

"a"는 'a'와 뒤에 null문자가 같이 존재한다. (크기 2의 char 배열이다)

 

파이썬에서는 둘의 차이가 없다.

 

파이썬 수업듣다가 작은따옴표 큰따옴표를 찾아봤는데

c,c++에서 작은따옴표가 한글자, 큰따옴표가 string인건 알았는데 큰따옴표로 쓰면 뒤에 null이 있는건 처음알았다.
