dp 문제. 

 

// times[i] = j : i초에 j 위치에 자두 떨어짐
int times[1010];
// d[i][j] : j초 까지 총 i번 움직였을때 {현재 위치, 먹은 자두 최댓값}
pair<int,int> d[35][1010];

 

각 초에 대하여 현재까지 총 몇번 움직였고, 현재 내가 몇번 위치에 있고, 총 먹은 자두의 갯수를 기억하면 된다.

 

생각해야 하는 경우는 총 두가지이다.

현재 초 (second)에서 다음 초로 넘어갈때 움직일것인가 움직이지 않을 것인가. 

 

움직인다면 현재 위치가 다른 위치로 바뀔것이고, 바뀐 위치가 다음 초에 자두가 떨어지는 위치와 일치한다면 현재까지 먹은 자두의 갯수가 1 증가한다.

 

움직이지 않는다면 현재 위치가 유지될것이고, 다음 초에 자두가 떨어지는 위치와 일치한다면 현재까지 먹은 자두의 갯수가 1 증가한다. 

 

위 두 경우 중에 자두의 갯수가 더 많은쪽을 기억하면 된다.
