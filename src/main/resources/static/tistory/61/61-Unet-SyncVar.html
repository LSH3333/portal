SyncVar은 서버에서 클라이언트의 방향으로만 이루어진다.

서버에서 변수가 수정되면 모든 클라이언트로 동기화,

클라이언트에서는 이 변수의 값을 수정해도 다른 클라이언트나 서버에서는 변경이 되지 않음.

 

hook은 서버에서 변경된 값이 클라이언트에 통지되었을때 동작하는 것이므로, 클라이언트에서만 동작한다.

또한, hook을 사용하면 그냥 SyncVar를 사용했을 때와 다른것이 클라이언트의 맴버 변수에 동기화되어야 하는 값이 바로 적용되지 않는다. hook으로 등록한 함수의 매개변수를 통해서 전달되므로, 클라이언트 측의 맴버 변수에 매개변수의 값을 직접 대입해줘야 한다.
