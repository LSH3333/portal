평소와 같이 깃허브에 푸쉬를 하는데 다음과 같은 에러가 떴다.

 

원인은 git CLI 때문이었다.

저번에 CLI에 내 토큰을 저장해 놓았었는데

어제였나 메일로 토큰 만료된다고 연장하라고 해서 아무 생각 없이 연장했었는데 

이때 토큰이 regenerate 되면서 변경되었고

CLI에 저장해놓은 내 토큰이랑 불일치해서 생긴 오류였다.

 

다음과 같이 CLI에 로그아웃하고, 다시 로그인해서 새로운 토큰을 입력해서 해결.

gh auth logout 

gh auth login
