a[][] : 주어지는 숫자

d[][] : [1][1] ~ [r][c] 까지의 수의 합 

 

우선 2차원 배열 d에 d[r][c] = [1][1] ~ [r][c] 까지의 수의 합을 구한다.

d[r][c] = d[r-1][c] + d[r][c-1] - d[r-1][c-1] + a[r][c];

 

주어지는 r1, c1, r2, c2 에 대하여 

[r1][c1] ~ [r2][c2]까지의 합은  = d[r2][c2] - d[r2][c1-1] - d[r1-1][c2] + d[r1-1][c1-1]
