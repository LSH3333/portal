순열과 조합을 만드는 법을 재대로 공부하고 나니까 

대부분의 brute force 문제가 훨씬 편해졌다. 

감사합니다 permutation.

 

n개의 숫자중 부분수열의 합이 s가 되는 부분수열의 갯수를 구하는 문제.

n개의 숫자중 1~n개를 뽑는 조합을 만들어 그 합을 구하는 방식으로 풀었다.

 

n=5이고 주어진 수열이 {-7, -3, -2, 5, 8} 이라면 

5개중 1개를 뽑는 조합 =

{-7}, {-3}, {-2}, {-5}, {8}

 

5개중 2개를 뽑는 조합 =

{-7,-3}, {-7,-2}, {-7,5}, ... {5,8}

 

이런식으로 1개부터 n개를 뽑아 조합을 만들어 그 조합의 숫자를 모두 더해서 s가 되는 부분수열을 구했다.

 

#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int n, s;
vector<int> v;
// permutation을 위한 인덱스 
vector<int> num_idx;
int ans = 0;

int main() {
    cin >> n >> s;
    for(int i = 0; i < n; i++) {
        int num;
        cin >> num;
        v.push_back(num);
    }

    int temp = 1;
    while(temp <= n) {
        // permutation 돌리기위해 temp개 만큼 1삽입. 나머진 0으로 채움. 
        // 1 삽입
        for(int i = 0; i < temp; i++) {
            num_idx.push_back(1);
        }
        // 0 삽입
        for(int j = 0; j < n-temp; j++) {
            num_idx.push_back(0);
        }
        // n개의 숫자중 temp개를 뽑는 조합 
        do {
            int sum = 0;
            for(int i = 0; i < n; i++) {
                if(num_idx[i] == 1) {
                    sum += v[i]; // 만들어진 조합들을 더함 
                }
            }
            // 더한결과가 s라면 찾는 부분수열임 
            if(sum == s)
                ans++;
        } while(prev_permutation(num_idx.begin(), num_idx.end()));

        num_idx.clear();
        temp++;
    }

    cout << ans << '\n';

}
