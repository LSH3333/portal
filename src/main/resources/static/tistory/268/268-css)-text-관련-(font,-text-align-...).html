font-stack

내가 지정한 폰트를 브라우저가 지원하지 않을때를 대비해 여러개의 폰트를 설정해놓음

이렇게 여러개의 font를 지정할수 있고, 모든 폰트를 지원하지 않으면 마지막에 generic font 중에서 하나를 택하도록함.

 

generic fonts : serif, sans-serif, cursive, fantasy, monospace

generic fonts는 다섯가지가 있고, 모든 font들은 이 generic fonts 중 하나에 속함.
