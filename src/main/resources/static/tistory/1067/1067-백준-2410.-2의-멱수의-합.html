2차원 배열로 푸는 방법

푸는 방법은 d 배열을 아래와 같이 정의했다.

// d[i][j] = i를 2의 멱수의 합으로 나타내는 경우의 수 중 뒤가 pow(2,j) 로 끝나는 경우의 수
int d[MAX+10][22];

예를들어 6을 만드는 경우는 아래와 같이 된다. 

 

d[6][0] = 1+1+1+1+1+1   

d[6][1] = 1+1+1+2, 1+2+2, 2+2+2

d[6][2] = 1+1+4, 2+4 

 

 

 

1차원 배열로 푸는 방법

1차원 배열로 푸는 방법은 

1로 만드는 방법을 1~N까지 모두 계산하고

1,2로 만드는 방법을 1~N까지 모두 계산하고 

1,2,4로 만드는 방법을 1~N까지 모두 계산 ..

 

1차원 배열로 푸는 이 방법은 계산 순서를 바꾸는 것이다. 

보통 생각할때 N=1에 대해서 계산하고, N=2 , 3 .. 이런 식으로 하나의 행을 모두 처리하고 다음 행을 계산하는 식으로 생각하게 되는데

이 방법은 하나의 열을 먼저 모두 계산하고 다음 열을 계산한다.
