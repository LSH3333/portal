작업중인 유니티 프로젝트 깃헙에 올리기

Git

2021-09-27 20:47:07

1. 깃헙에서 새로운 레포지토리 만든다.

이때 .gitignore에서 유니티를 추가한다

2. 유니티 프로젝트 폴더로 가서 git init으로 깃 스켈레톤 생성, 리모트 레포지토리 추가, 풀.

 

git init

git remote add [리모트명] https://github.com/계정명/레포지토리이름 

git pull origin main 

 

3. 폴더 전체 커밋, 푸쉬.

 

git add .

git commit -m "my commit"

git push
