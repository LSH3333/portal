오브젝트가 특점지정으로 고개를 돌려 이동

특점 지정으로 이동

// 현재 오브젝트가 moveSpot으로 서서히 이동한다
transform.position = Vector3.MoveTowards(transform.position, moveSpot.position, speed * Time.deltaTime);

 

이동할때 오브잭트의 고개가 돌아가도록 하려면?

예를들어 오브젝트가 사람이라면 특점 지정으로 이동할때 그 지점으로 고개가 돌아가야 자연스럽다

// 이동하는 지점으로 고개를 돌림
transform.rotation = Quaternion.Slerp(transform.rotation, Quaternion.LookRotation(moveSpot.position), 0.15f);

Quaternion.Slerp와 Quaternion.LookRotation을 이용.
