요즘 재귀함수 만드는걸 열심히 한 성과가 있었는지 재귀를 이용해서 금방 풀수있었다.

재귀가 너무 어렵고 스트레스 받았는데 드디어 좀 익숙해진것 같다.

 

1. 화단의 [1][1] 지점 부터 [n-1][n-1] 까지 탐색하면서 그 좌표를 중심점으로 하는 꽃을 심을수 있는지 확인한다.

   [0][0] 부터 [n][n]까지가 아닌 이유는 가장자리에 심으면 무조건 화단을 벗어나기 때문이다.

 

2. 심을수 있는 지점이면 심고 bool mark[][] 배열에 심었다고 기록한다.

 

3. 재귀함수의 형태로 3개의 꽃을 다 심으면 3개의 꽃이 심어진 상태에서 가격이 얼마인지 계산하고 최솟값을 갱신한다. 

 

4. 되돌아가서 마지막에 심은 꽃을 파내고 다른 지점에 심어본다.

 

 

#include <iostream>
#include <algorithm>

using namespace std;
#define MAX 10

int n;
int ans = 999999999;
int map[MAX][MAX];
bool mark[MAX][MAX];


// map[r][c]를 중심으로 하는 꽃을 심을수 있는지 확인
bool CanIPlantFlowerHere(int r, int c)
{
    // 화단을 벗어나면 심을수 없음
    if(r-1 < 0 || r+1 >= n) return false;
    if(c-1 < 0 || c+1 >= n) return false;

    // 꽃이 이미 심어져있다면 심을수 없음
    if(mark[r][c] || mark[r-1][c] || mark[r][c+1] || mark[r][c-1] || mark[r+1][c])
        return false;

    // 모든 조건 통과 = 꽃 심을수 있음
    return true;
}

// 꽃을 심어본다
void PlantFlower(int r, int c)
{
    mark[r][c] = true;
    mark[r-1][c] = true;
    mark[r][c+1] = true;
    mark[r][c-1] = true;
    mark[r+1][c] = true;
}

// 꽃을 파낸다
void DigFlower(int r, int c)
{
    mark[r][c] = false;
    mark[r-1][c] = false;
    mark[r][c+1] = false;
    mark[r][c-1] = false;
    mark[r+1][c] = false;
}

// 3개의 꽃이 심어진 상태에서 가격이 얼마인지
int CalculatePrice()
{
    int price = 0;
    for(int i = 0; i < n; i++)
    {
        for(int j = 0; j < n; j++)
        {
            if(mark[i][j])
                price += map[i][j];
        }
    }
    return price;
}

// 모든 꽃을 심을수 있는 경우를 탐색한다
void Flower(int depth)
{
    if(depth == 3)
    {
        ans = min(ans, CalculatePrice());
        return;
    }

    for(int r = 1; r < n-1; r++)
    {
        for(int c = 1; c < n-1; c++)
        {
            if(mark[r][c]) continue;

            // 해당 지점에 꽃을 심을수 있으면 심는다
            if(CanIPlantFlowerHere(r,c))
            {
                PlantFlower(r,c);
                Flower(depth+1);
                DigFlower(r,c);
            }
        }
    }
}

int main()
{
    cin >> n;
    for(int i = 0; i < n; i++)
    {
        for(int j = 0; j < n; j++)
        {
            cin >> map[i][j];
        }
    }

    Flower(0);

    cout << ans;


}
