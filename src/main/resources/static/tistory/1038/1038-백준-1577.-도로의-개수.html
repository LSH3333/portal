백준 1577. 도로의 개수

2023-01-27 16:10:08

이런식으로 도로가 격자형태로 있을때  최단 거리 dp 문제의 점화식은 보통 d[r][c] = d[r-1][c] + d[r][c-1] 이다.

그런데 이 문제는 이동 하지 못하는 도로가 있다.

이 문제의 핵심은 이동 못하는 도로를 처리하는 것이다.

(그리고 이 문제는 N이 가로 M이 세로로 주어지고 좌표도 좀 햇갈리게 주어지는데 어처피 경우의 수를 구하는 거라 계산하기 편하게 배열처럼 바꿔줘도 상관없다) 

 

[r][c]에서 위,오른쪽,아래,왼쪽 도로가 이동 가능한지 아닌지를 표시해놓는다. 

이렇게 하면 점화식  d[r][c] = d[r-1][c] + d[r][c-1] 에서 만약 d[r-1][c]는 위에서 현재 지점으로 오는 경우고, d[r][c-1]은 왼쪽에서 현재 지점으로 오는 경우인데, 만약 위쪽 도로가 막혔으면 위에서 현재 지점으로 오는 경우는 더하지 않고, 왼쪽 도로가 막혔으면 왼쪽에서 현재 지점으로 오는 경우를 더하지 않으면 된다. 

 

막힌 도로가 어느 방향인지 판단은 좌표 (r1,c1) , (r2,c2) 가 있을때 r+c가 큰 좌표가 오른쪽 아래에 가깝다.

그리고 r1==r2 라면 열이 다른 열이고, c1==c2 라면 행이 다른 행이다.
