보통 유니티 폴더를 깃헙에 올릴때 용량문제로 .gitignore로 

Library, Log폴더 같은것들은 제외 시킨다.

이렇게 깃헙에 올라간 상태에서

 

다른 컴퓨터에서 그대로 이 프로젝트를 실행시키려면

1. 먼저 같은 프로젝트 이름으로 프로젝트를 만든다.

   그럼 Library,Log를 포함한 폴더들이 자동 생성된다.

 

2. Assets, Package, ProjectSettings를 지운다.

 

3. git init으로 스켈레톤 만들고 remote 추가

 

4. git pull origin main
