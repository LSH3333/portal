다익스트라 알고리즘을 이용해 풀었는데 효율성을 모두 통과하지는 못했다.

조금 더 고민해봐야겠다.

코드:

 

 

다익스트라로 효율성까지 모두 맞는 방법을 찾았다.

이전에 내 코드는 i번 정점까지 합승하고, i번부터 각각의 정점까지 최소비용을 다익스트라 알고리즘으로 찾았다.

즉 다익스트라를 n번 수행했다.

 

하지만 생각해보면 모두 합쳐서 총 3번의 다익스트라로 풀수있다.

1. 시작지점 S에서 모든 지점까지의 최소비용 (즉 합승요금)

2. A지점에서부터 모든 지점까지의 최소비용 

3. B지점에서부터 모든 지점까지의 최소비용 

 

이렇게하면 (1번 합승비용) + (2번 A에서 모든 지점까지의 최소비용) + (3번 B에서 모든 지점까지의 최소비용)

중 최솟값이 답이다.
