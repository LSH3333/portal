dp, dfs 문제.

 

 

루트 노드 1 부터 시작해서 두가지 경우를 생각해야 한다.

현재 노드가 얼리 어뎁터인 경우, 아닌 경우.

 

우선 현재 노드가 얼리 어뎁터가 아니라면 인접 노드들은 모두 얼리 어뎁터이어야 하는 것은 확실하다.

현재 노드가 얼리 어뎁터인 경우에는 인접 노드는 얼리 어뎁터일수도 있고 아닐 수도 있다. 

 

최종적으로 dfs를 시작한 노드인 1의 값 d[1][0], d[1][1] 중 작은 값이 답이된다.
