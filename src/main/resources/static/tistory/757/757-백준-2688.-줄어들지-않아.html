d[i][j] : i 개 자릿수에서, j로 끝나는 숫자의 갯수 

 

점화식은 다음과 같다.

d[i][j] = d[i-1][0] + d[i-1][1] + ... + d[i-1][j] 

 

n=2 일때 마지막 숫자가 3인 숫자의 갯수는 몇개일까? (d[2][3]) 

마지막 숫자가 3이 되려면 이전 자릿수가 3보다 작거나 같은 숫자여야 한다. 

즉 d[2][3] =  d[1][0] + d[1][1] + d[1][2] + d[1][3]이 되는 것이다.
