백준 2239. 스도쿠

PS

2022-05-18 14:04:12

순열을 만들 듯이 스도쿠를 채워나가면 된다.

행과 열에 있는 숫자들은 따로 bool형 2차원 배열을 만들어 해당 행이나 열에 해당 숫자가 있으면 참이되도록 체크해줬다.

예를들어 R[10][10] 2차원 bool형 배열에서 R[3][2]가 참이라면 스도쿠의 3행에는 2가 이미 존재한다는 뜻.

3x3 사각형에 숫자가 있는지 확인은 그냥 그때 그때 확인해줬다.
