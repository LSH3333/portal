재귀로 풀어보았다.

별의 모양이 사각형 안에 사각형이 들어있는 모양인데

맨 안쪽 사각형 부터 생각하면 한변의 별의 갯수가 

n=1 일때 1개

n=2 일때 5개

n=3 일때 9개

 

즉 한변의 별의 갯수 = 4 x (n-1) + 1 이란것을 알 수 있다.

 

재귀함수에 n, 별찍기 시작할 좌표 r,c를 넘겨준다.

함수에 파라미터를 넘길때 n은 1씩 줄인다.

별 찍기 시작할 좌표 r,c는 첫 별을 사각형의 좌측상단으로 생각하면 

첫번째 사각형의 첫별 위치 (0,0)

두번째 사각형의 첫별 위치 (2,2)

세번째 사각형의 첫별 위치 (4,4) 로 2씩 늘어나므로 r+2, c+2를 넘겨주면 된다.

 

#include <iostream>
using namespace std;

char stars[500][500];

void print(int n, int r, int c) {
    if(n == 0) return;

    int len = 4 * (n-1) + 1;
    //cout << "n: " << n << ' ' << "(" << r << "," << c << ") " << "len: " << len << '\n';
    for(int i = r; i < r+len; i++) {
        for(int j = c; j < c+len; j++) {
            // 첫행과 마지막행일 경우, len개의 * 출력
            if(i == r || i == r+len-1) stars[i][j] = '*';
            else { // 그 외에는 첫열과 마지막열에만 * 출력
                if(j == c || j == c+len-1) stars[i][j] = '*';
                else stars[i][j] = ' ';
            }
        }
    }

    print(n-1, r+2, c+2); // 재귀 

}

int main() {
    int n;
    cin >> n;
    int len = 4 * (n-1) + 1;

    print(n,0,0);

	// 별 출력 
    for(int i = 0; i < len; i++) {
        for(int j = 0; j < len; j++) {
            cout << stars[i][j];
        } cout << '\n';
    }
}
