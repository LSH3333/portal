브루트포스로는 안될꺼고, map 에 S의 모든 접두를 저장하는 방법이 될지 안될지 확신이 안섰는데 일단 해봤다.

(트라이 인것 같긴한데 트라이 짜기 힘들어서 일단 map으로 해봤다..)

 

그리고 통과했다. 

 

 

그리고 나서 찾아 보니까 역시 정식 풀이 방법은 아니었다 (cpp 말고 파이썬 같은 언어로는 tle ) 

 

 

 

이분탐색으로 풀수 있다 길래 이분탐색으로도 풀어 봤다.

이분탐색으로 푸는 방법은 집합 S 를 정렬해서 이분탐색 하면 된다.

주어진 문자열과 집합 S에 속한 문자열을 비교해서 left, right 를 조절하면 된다.

 

예를들어 S 가 다음과 같고 

 

baekjoononlinejudge
codeplus
codingsh
startlink
sundaycoding

 

접두열인지 탐색하는 문자열이 baekjoon 이다.

처음에 mid는 가운데인 codingsh 이다.

baekjoon 과 codingsh 를 비교하면 b와 c 가 다르고, b가 c 보다 작기 때문에 right = mid - 1 이 된다.
