누적합 문제.

 

'a' ~ 'z' 까지 주어진 문자열 S의 i번 인덱스 까지 해당 알파벳이 몇번 등장했는지 누적합으로 구한다.

mark[10]['a'] = 3 이라면 주어진 문자열 S에서 10번째 인덱스까지 'a'가 3번 등장했다는 의미다.

 

이렇게 구해놓고나면 a 2 5, 즉 [2,5]까지 'a'가 몇번 등장했는지 계산은 mark[3]['a'] - mark[2-1]['a'] 로 하면된다. 

2포함에서 ~ 5 포함까지 이므로 2-1인 mark[1]['a]를 빼줘야한다.

 

주의할점은 만약 a 0 3 이런식으로 시작 인덱스가 0일때는 예외처리해줘야 한다.
