지나온 도시중에서 기름이 가장 싼 가격을 기억해놨다가 

도시를 방문할때마다 가장싼가격 x 다음도시까지거리 

아주 전형적인 그리디문제 

 

제일 왼쪽 도시부터 제일 오른쪽 도시까지의 거리는 1이상 1,000,000,000 이하의 자연수.

리터당 가격은 1 이상 1,000,000,000 이하의 자연수

이기 때문에 입력을 저장하는 배열은 int형이어도 상관없지만

곱한 결과가 저장되는 ans와 곱하는 대상이되는 minCost는 long long으로 선언해줘야 한다.

 

 

#include <iostream>
using namespace std;

int n;
long long ans = 0;
int road[100001];
int cost[100001];

int main()
{
    cin >> n;
    for(int i = 0; i < n-1; i++)
        cin >> road[i];
    for(int i = 0; i < n; i++)
        cin >> cost[i];

    long long minCost = 2000000001;
    for(int i = 0; i < n-1; i++)
    {
        // 현재까지 가격중 가장 낮은 가격으로 계속 주유한다
        if(cost[i] < minCost)
        {
            minCost = cost[i];
        }

        // 가장 낮은 가격 * 다음 도시까지 거리
        ans += minCost * road[i];

    }
    cout << ans;
}
