처음 시도한건 큐를 페어형으로 만들어서 (트럭무게, 현재지점) 을 큐에넣어 계산하는 방법이었다.

그런데 하다가 더 좋은 방법이 생각났다. 큐 자체에 크기를 제한시키는 방법이다.

 

truck_weight를 큐(trucks)에 저장해놓는다.

또다른 큐(q)에 bridge_length개 만큼 0을 푸쉬해놓는다. 큐의 크기를 bridge_length개로 제한하기 위해서다.

 

trucks = {7,4,5,6}

q:

0s 0 0
1s 0 7
2s 7 0
3s 0 4
4s 4 5
5s 5 0
6s 0 6
7s 6 0
8s 0 0

 

q의 front를 계속 pop하면서 (다리에서 내리면서)

현재 다리의 무게 + 다음 트럭의 무게가 다리가 견딜수 있는 무게보다 크다면 0을 q에 푸쉬하고

작다면 다음 트럭을 q에 푸쉬한다. 푸쉬할때마다 현재 무게를 더해준다.
