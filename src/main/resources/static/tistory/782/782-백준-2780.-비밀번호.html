백준 2780. 비밀번호

PS

2022-07-14 16:26:29

https://www.acmicpc.net/problem/2780

 

d[i][j] = 길이가 i면서 숫자 j로 끝나는 비밀번호의 갯수

 

끝이 x로 끝나는 번호는, 끝 이전 번호가 x와 인접한 수여야 한다.

예를들어 d[2][1]은 길이가 2 면서 끝이 1인 비밀번호다. 

끝이 1이 려면 끝 이전 번호가 2 혹은 4여야 할것이다.

따라서 d[2][1] = d[1][2] + d[1][4] 

 

a[j] = {j와 인접한 숫자} 일때, 

점화식 : d[i][j] = d[i-1][a[j][0]] + d[i-1][a[j][1]] ...

 

 

주의할점:

더할때 MOD로 나눈 나머지를 다 더하고, 마지막에 답을 출력할때 % MOD를 안해서 틀렸었다.
