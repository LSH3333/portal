Meshrenderer로 색상 컨트롤.

코드를 이용해 색 변경:

public class ColorControl : MonoBehaviour 
{
	public MeshRenderer _renderer;
    
    void Start()
    {
        if (isLocalPlayer) _renderer.material.color = Color.blue;
        else _renderer.material.color = Color.red;
    }
}

 

Material을 이용:

 

Create - Material로 Material 생성하고, Albedo 에서 색상 변경.

그 후 색상변경 원하는 오브젝트의 Mesh Renderer의 Materials의 Element에 드래그.
