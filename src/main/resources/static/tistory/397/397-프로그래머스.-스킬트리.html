skill의 각 알파벳마다 순서대로 번호를 매긴다.

skill이 "CBD"라면 C=1, B=2, D=3. 없는 알파벳은 0으로.

int alp[26] = {0,};
int cnt = 1;
for(int i = 0; i < skill.size(); i++)
{
alp[skill[i]-'A'] = cnt++;
}

 

skill_trees에서는 무조건 1번 알파벳 부터 순서대로 나와야한다.

2번 알파벳 혹은 3번 알파벳부터 나왔다면 1번 알파벳 스킬을 안배웠기 때문에 불가능하다.

또한 0인 알파벳은 상관없는 스킬이므로 무시해주면 된다.
