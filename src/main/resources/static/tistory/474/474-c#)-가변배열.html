c#은 특이한 배열이 있는데 바로 가변배열이다.

가변배열은 다차원 배열과는 조금 다르다.

 

c#의 다차원배열 

// 2차원 배열            
int[,] arr = new int[,]
{
	{1,2 },
	{3,4 }
};

이건 다른 언어들의 일반적인 다차원 배열이다.

 

 

c#의 가변배열

가변배열은 각 차원의 길이를 다르게 만들수 있다.

jaggedArr 배별은 0행은 크기가 2고, 1행은 크기가 3인것을 볼 수 있다. 

 

가변배열은 배열을 요소로서 갖는 배열로서, 배열을 요소처럼 접근할수 있다.
