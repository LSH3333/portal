오름차순 정렬후 실제등수와 예상등수의 차의 절댓값을 더해나가면 되는 문제.

정답의 자료형은 long long으로 선언해줘야한다.

 

#include <iostream>
#include <algorithm>
using namespace std;

int n;
long long arr[500002];

int main()
{
    cin >> n;
    for(int i = 1; i <= n; i++)
        cin >> arr[i];

    sort(arr+1, arr+n+1);

    long long ans = 0;
    for(int i = 1; i <= n; i++)
    {
        ans += abs(arr[i]-i);
    }

    cout << ans;

}
