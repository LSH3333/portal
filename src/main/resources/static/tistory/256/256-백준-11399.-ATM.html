앞에 있는 사람이 걸리는 시간만큼 계속 누적해서 더하므로

앞에 시간이 작은 사람이 오는것이 유리하다.

 

오름차순 정렬 후

누적시간을 계속 더해주기만 하면 되는 문제.

 

이문제도 문제에서 오름차순된 예를 보여주고 이 예의 답이 최소시간 이라는 것을 알려주기 때문에 답자체를 구하기는 쉽다.

 

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

vector<int> v;
int n;

int main()
{
    cin >> n;
    for(int i = 0; i < n; i++)
    {
        int num;
        cin >> num;
        v.push_back(num);
    }

    // 오름차순 정렬
    sort(v.begin(), v.end());

    int ans = 0;
    int time = 0;
    int prev = 0;
    for(int i = 0; i < n; i++)
    {
        prev = time;
        time = prev + v[i];
        ans += time;
    }
    cout << ans;
}
