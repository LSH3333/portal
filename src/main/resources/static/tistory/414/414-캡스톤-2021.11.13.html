사용자가 원하는 이미지로 수정하는 기능은 구현 완료.

텍스트 수정에 아직 문제가 있다.

지금 구현된 기능으로는 사용자가 웹상에서 텍스트 파일을 일일히 직접 지정해줘야하는데

너무 번거로울뿐만 아니라 나중에 만약에 서버에 올리는 상황이 오면 문제가 있을것으로 예상된다. 

따라서 다른 방법이 필요할것 같다.

 

생각해놓은 방법으로는 html 파일의 모든 문자열을 불러와서 유니티 스크립트에서 그 문자열을 수정한후 html 파일에 반영하는 것.

그러기 위해서 먼저 유니티상에서 html 파일을 불러오고 수정하는 기능이 정상적으로 작동되는지 테스트가 필요하다. 

 

 

- Unity에서 html 파일 불러오는 기능 확인 

 

- Unity에서 html 파일 수정되는지 확인 

 

 

 

1. html 파일 읽어온다

2. 읽어온 html 파일을 문자열로 모두 바꾼후에 각 라인별로 "FixFlag" 라는 문자열이 포함되어있는지 확인한다

("FixFlag"는 이 부분의 html 코드가 변경이 필요하다는 기준점) 

3. "FixFlag"가 포함되어있으면 다음 줄에 유저가 지정한 문자열들을 차례대로 삽입한다.

 

 

다음 버튼을 누르니 index.html 파일이 정상적으로 생성된것을 볼 수 있다.
