ManyGames REST 서버

  • ManyGames의 데이터 저장을 위한 DB 와 유니티 클라이언트 사이의 서버입니다.
  • 원래 NCMB 라는 back end as service 로 데이터를 저장하고 있었는데 서비스가 종료 예정이라는 통보를 받고 새로운 DB 가 필요했습니다.
  • 서버는 REST api 를 구현해 유니티 클라이언트에서 HTTP 요청을 보내면 경로에 따라 데이터를 처리하고 DB 와 소통합니다
  • JPA로 객체와 테이블을 매핑해서 소통했습니다.

유니티 클라이언트
