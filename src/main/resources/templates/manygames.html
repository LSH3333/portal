DB

DB는 두가지를 교체가능하도록 구현했습니다.

  • NCMB (back end as service)
    • nifcloud 사에서 제공하는 baas
    • 클라이언트 키로 유니티 클라이언트와 baas 를 연결해서 데이터 저장 가능
  • REST 서버
    • 유니티 클라이언트에서 데이터 처리 HTTP 요청을 보내면 DB와 소통하는 서버

여러가지 미니 게임들 통합 관리

  • 점수 저장, 결과창 등 모든 미니게임들에서 공통으로 쓰는 로직 Manage 클래스에 추상적으로 구현
  • DB 서버와 소통도 Manage 클래스에서 담당하기 때문에 DB 교체 시 Manage 만 변경
